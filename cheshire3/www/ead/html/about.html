About Cheshire for Archives

Cheshire for Archives is an application of the Cheshire3 Information Framework. It provides a sophisticated search engine for EAD encoded archival finding aids, as well as tools to administer the database via a web-interface.

New Features!
As of version 3.4 there's even a data creation and editing interface! Based loosely on the Archives Hub data creation template, this new interface allows you to create new catalogue entries from scratch and insert them directly into the database - there's no need to worry about tags, schemas, DTDs or any of that stuff! You can even load one of your existing files into the interface to correct spelling mistakes, or even catalogue additional material within the collection. For more details see the creation and editing template help page.

As well as a human useable web-interface for searching, it also exposes the database via web services (web services are the cornerstone of Web 2.0!)
